Install the project an perform the following steps

1. Set PYTHONPATH=EezzServer/webroot/applications/EezzServer
2. Execute 
    pthon server.py --webroot=<path>/EezzServer/webroot --host=localhost --websocket=8000

3. Start the browser
    http://localhost:8000/esptest.html
    http://localhost:8000/esptree.html

Prerequisits:
    Install pycrypto --py3.3 e.g. from http://www.voidspace.org.uk/python/modules.shtml#pycrypto
